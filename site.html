css yazma yöntemleri : 1-)dosya içinde yazma : head alanında style etiketi arasında tanımlanabilir. 2-) site.css adında bir sayfa açıp orada tanımlanabilir.Bu aynı zamanda karışıklığı önler. bunun için head alanında site.css için link vermek gerekir.

seçiciler : html tarafından stil vermek istediğimiz elemanları seçme şekilleri. type selector : elementname örnek olarak bu seçme şekli site.cssde h1'i seçme şeklimizdir. aynı şekilde p elementine (ve diğer elementlere) de renk verebiliriz.

universal selector : * yıldız ile tanımlanır. * sayfada head alanı dışında her yeri kapsar ve tanımlanan stili uygular.

* ile stil verdikten sonra örnek olarak type selector ile tekrar stil verirsek , son yazdığımız type selector ile verdiğimiz stil geçerli olur. çünkü kodlar yukarıdan aşağı okunmaya başladıktan sonra son okunan geçerli olur.

class selector : .classname (ul sırasız listede kulanılır) we created a list and added classes. then we named classes. 1-) we can add style to li tag. 2-) we can add style to classes.
